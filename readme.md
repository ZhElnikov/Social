#Технологии:
-Apache-2.4
-PHP-5,6
-MySQL-5.5
-Laravel-5.2

#Помощь по установке

-открыть консоль
-перейти в domains
-прописать команды
-git clone https://github.com/ZhElnikov/Social.git
-перейти cd laravel
-composer self-update
-composer install
-composer self-update
-создать .env (переименовать .env.example)
-создать базу (с названием проекта(папки))
-подключить базу в .env
-php artisan migrate
-php artisan db:seed
